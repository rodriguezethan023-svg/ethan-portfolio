Case Study_02 // Retail Analytics

Anime Paradise Analytics

For the Anime Paradise retail project, I engineered an end-to-end sales dashboard to track global performance metrics. This visualization system allows stakeholders to monitor real-time KPIs, inventory levels, and customer demographics through a high-contrast, branded reporting interface.

The Methodology

I utilized DAX to create dynamic time-intelligence measures, allowing leadership to switch between YoY, QoQ, and MoM performance views instantly. The UI was customized using Figma and Canva to ensure the analytics felt like an extension of the Anime Paradise brand.

Visual Breakdown

Incorporated cross-filtering slicers, drill-through actions, and custom tooltips to provide a deep dive into product-level profitability without cluttering the main executive view.
